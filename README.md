## Resources
Currently available resources for exploring the data dictionary

- Public link to explore the ABCD data dictionary: https://abcd.deapscience.com/#/my-datasets/create-dataset
- Public link to download the ABCD data dictionary (needs the NBDCtools R package): https://software.nbdc-datahub.org/NBDCtools/reference/get_dd.html
- Actual data (dictionary) available in a public repository: https://github.com/nbdc-datahub/NBDCtoolsData/tree/main/data
- Reference for the above data: https://software.nbdc-datahub.org/NBDCtools/reference/get_dd.html
