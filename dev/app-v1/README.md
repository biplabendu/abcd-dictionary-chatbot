# Running it on your computer:

* Clone the repo.

* Open the project in RStudio (renv will automatically download the right R packages based on renv.lock).

* Run the terminal commands to activate the virtual environment and install required python libraries
    * `python3 -m venv python_env`  
    * `pip install -r requirements.txt`

In RStudio,
    * Open the app.R file
    * Click "Run App"
